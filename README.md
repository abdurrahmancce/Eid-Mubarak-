# 🎆 Eid Fireworks & Greetings 🎉

Celebrate Eid with a colorful terminal animation!  
This Python project brings joy to your console with fireworks, festive ASCII art, and heartfelt wishes.

---

## ✨ Features
- **Fireworks Animation**: Randomly colored festive symbols (`🎈 🎆 ✨ 🎉 🎊`) light up your terminal.
- **Eid Mubarak Text**: Animated ASCII art using `pyfiglet` with vibrant colors.
- **Blessing Message**: A final heartfelt wish displayed in cyan.
